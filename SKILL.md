# Web Guide - 浏览器自动化操作指导系统

## 核心功能

1. **逐步指导**: 引导用户完成浏览器自动化操作的每一步
2. **经验归档**: 成功后自动总结操作流程为固定方法
3. **方法复用**: 归档的方法可被自动调用实现自动化重复操作

## 工作流程

### 首次操作（引导模式）

当用户描述一个浏览器自动化任务时：

1. **意图解析**: 分析用户需要完成的操作
2. **步骤规划**: 将操作拆解为具体的 Playwright 操作步骤
3. **执行确认**: 逐步执行并让用户确认结果
4. **成功归档**: 操作成功后，将流程保存到 `methods/` 目录

### 后续操作（自动模式）

当用户再次请求相同任务时：

1. **关键字匹配**: 根据用户输入匹配已归档的方法
2. **自动执行**: 直接调用归档的方法脚本执行
3. **无需指导**: 跳过逐步指导，直接运行

## 关键词触发

当用户提到以下内容时自动触发：
- "帮我操作浏览器"
- "自动化操作网站"
- "登录网站并..." (需要浏览器操作的场景)
- "点击网页上的..."
- "提取网页数据"
- 任何需要 Playwright/浏览器自动化的场景

## 方法归档格式

归档的方法存储在 `methods/` 目录下，以 JSON 格式保存：

```json
{
  "name": "方法名称",
  "keywords": ["触发关键字1", "触发关键字2"],
  "description": "方法描述",
  "createdAt": "创建时间",
  "lastUsedAt": "最后使用时间",
  "useCount": "使用次数",
  "script": "执行脚本路径",
  "steps": [
    {
      "action": "goto",
      "target": "https://example.com",
      "description": "打开网站"
    }
  ]
}
```

## 脚本说明

- `scripts/guide-runner.js` - 引导模式核心脚本，逐步执行操作
- `scripts/method-executor.js` - 自动模式执行器，运行归档方法
- `scripts/method-archiver.js` - 方法归档器，保存成功流程

## 使用示例

### 引导模式（首次操作）

```
用户: 帮我登录 jif8.net 并获取 V2Ray 订阅链接
系统: 正在规划操作步骤...
      1. 打开 Chrome 浏览器
      2. 访问 jif8.net/user
      3. 填写邮箱密码
      4. 点击登录
      5. 滚动到页面底部
      6. 点击复制 V2Ray 订阅
      7. 提取订阅链接
系统: 开始执行第 1 步：打开浏览器...
      [等待用户确认]
```

### 自动模式（后续操作）

```
用户: 获取 V2Ray 订阅
系统: 找到已归档方法: jif8-v2ray-subscribe
      正在自动执行...
      ✓ 浏览器已启动
      ✓ 已登录 jif8.net
      ✓ 已获取订阅链接
      订阅链接: https://xxx.example.com/xxx
```

## 注意事项

- 引导模式会逐步询问用户确认每一步操作
- 归档时自动提取可配置的账号密码等信息
- 方法归档支持版本管理，重复执行成功会更新方法
- 浏览器权限弹窗需要用户手动确认
